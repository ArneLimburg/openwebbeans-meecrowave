== Source code

Source code can be found at http://svn.apache.org/repos/asf/openwebbeans/meecrowave/trunk/

[source]
----
svn co http://svn.apache.org/repos/asf/openwebbeans/meecrowave/trunk/ meecrowave
----

=== Build it

To build the project just run maven:

[source]
----
mvn clean install
----
